# Halbautomatische Hobbygärtnerhilfe
Eine Hilfe für Hobbygärtner, die zu wenig Zeit haben.
Diese Hilfe gießt automatisch die Pflanzen und misst Temperatur, Luftfeuchtigkeit, Luftdruck und Bodenfeuchtigkeit.
